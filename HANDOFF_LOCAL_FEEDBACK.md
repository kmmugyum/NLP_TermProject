## 연구서버에 요청 (STEP 5~6 위임)
로컬 맥은 GPU 없음(torch CUDA=False, faiss 미설치) → 벡터 재빌드 불가.
**연구서버가 GPU(cuda:1)로 아래 수행 후 push 요청:**

1. `module1_indexer.build_vector_db()`로 `_academic_chunks_v2.json` → 새 `academic_real.bin`
   + `academic_real.bin.meta.json` 재빌드. KURE-v1, cuda:1 정책.
2. 빌드된 `.bin` + `.meta.json`을 repo에 push (또는 zip — .bin은 .gitignore이므로
   zip 별도 + 위치 보고. meta.json도 대용량이면 zip).
3. eval: 졸업이수학점/미적분학년 등 환각 질문으로 재빌드 인덱스 정확도 확인.
   환각 안 잡히면 push 보류하고 이 repo에 피드백.
4. v2 청크(`_academic_chunks_v2.json`)는 이미 push됨 → 교체 시
   `_academic_chunks.json`으로 rename.
